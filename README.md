## Quick Setup

1. Add `nuxt-yandex-pay` dependency to your project

```bash
# Using pnpm
pnpm add -D nuxt-yandex-pay

# Using yarn
yarn add --dev nuxt-yandex-pay

# Using npm
npm install --save-dev nuxt-yandex-pay
```

2. Add `nuxt-yandex-pay` to the `modules` section of `nuxt.config.ts`

```js
export default defineNuxtConfig({
  modules: [
    'nuxt-yandex-pay'
  ]
})
```

That's it! You can now use Nuxt Yandex Pay in your Nuxt app âœ¨

## Development

```bash
# Install dependencies
npm install

# Generate type stubs
npm run dev:prepare

# Develop with the playground
npm run dev

# Build the playground
npm run dev:build

# Run ESLint
npm run lint

# Run Vitest
npm run test
npm run test:watch

# Release new version
npm run release
```
